## Usage

1) Download a checkpoint

Look here: https://github.com/egorsmkv/ukrainian-flowtron-tts/releases

2) Do inference

```
python3 inference.py --eng 0 --sigma 0.5 -c config.json -f model_101000.pt -w waveglow_256channels_ljs_v3.pt -t "–í—ñ—Ç–∞—é –≤–∞—Å, –º–µ–Ω–µ –∑–≤–∞—Ç–∏ –õ–∞–¥–∞" -i 0
```
